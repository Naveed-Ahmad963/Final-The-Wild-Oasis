#  The Wild Oasis

The Wild Oasis is a modern hotel management web application built with **React**. It provides hotel staff with an intuitive dashboard to manage cabins, bookings, guests, check-ins, check-outs, and daily hotel operations efficiently.

##  Features

-  Secure Authentication
-  Cabin Management
-  Booking & Reservation Management
-  Guest Management
-  Check-in & Check-out System
-  Dashboard with Business Statistics
-  Booking Search & Filtering
-  Fully Responsive Design
-  Fast Data Fetching & Caching

##  Tech Stack

- React
- JavaScript (ES6+)
- Supabase
- TanStack React Query
- React Router
- Styled Components
- React Hook Form
- React Icons
- Recharts
- date-fns
- Vite

##  Installation

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:5173
```
